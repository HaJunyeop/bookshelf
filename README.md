# 담 · 나의 서재

책장 사진 23장을 바탕으로 만든 개인 도서 데이터베이스입니다. 현재 94권을 등록했고, 자동 판독이 불확실한 항목은 `검토 필요`로 분리했습니다.

## 가장 쉬운 사용법

1. 웹 화면에서 책을 검색하거나 장르로 거릅니다.
2. 별을 눌러 별점을 저장합니다.
3. `숨김`을 누르면 목록의 숨김 영역으로 이동하고 관계 지도에서도 사라집니다.
4. `관계 지도`에서 장르·저자·출판사·연도·언어·책장 위치를 선택합니다.
5. `추천 책`은 높은 별점의 장르를 우선해 새 책을 보여줍니다.
6. `CSV` 또는 `Obsidian` 버튼으로 현재 데이터를 내려받습니다.

별점과 숨김은 사용 중인 브라우저에 저장됩니다. 공개 사이트를 다른 기기에서 열면 기기별로 별도 저장됩니다.

## 새 사진 추가

1. `Photos` 폴더에 새 책장 사진을 넣습니다.
2. [책장 업데이트.cmd](책장%20업데이트.cmd)를 더블클릭합니다.
3. 새 사진만 무료 Windows OCR로 읽고, 확실하지 않은 항목은 검토 대상으로 추가합니다.

사진을 넣는 즉시 감시하려면 [사진 자동감시 시작.cmd](사진%20자동감시%20시작.cmd)를 실행한 채 창을 열어 둡니다.

## Obsidian

[ObsidianVault](ObsidianVault) 폴더를 Obsidian에서 `보관함으로 열기`로 선택합니다. 각 책은 독립된 Markdown 노트이며, `My Bookshelf.md`는 Dataview 플러그인이 있을 때 전체·숨김·검토 목록을 자동으로 보여줍니다. 관계 지도와 추천은 웹 화면을 사용하면 됩니다.

## 개인정보

원본 `Photos` 폴더와 OCR 원문은 GitHub에 올라가지 않도록 제외했습니다. 공개 페이지에는 정리된 책 메타데이터만 포함됩니다.
